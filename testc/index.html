Please update file "style.css" so that 3 cases below are satisfied:

  1. With screen >= 992px, display all divs with classB on 1 line (so in total, we have 1 lines with 9 cells)
  2. With screen >= 576px and <= 992px, display all divs that has classA on 1 line (so in total we have 3 lines, in each line we have 3 cells)
  3. With screen < 576px, display each div with classB in its own line (so in total we have 9 lines)

In all 3 cases, div with classB always fills as much space as it can (for example, in case 1, 9 cells will fill the screen's width, each cell is about 1/9 of the screen)
